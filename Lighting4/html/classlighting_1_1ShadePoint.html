Checks if the endpoints passed in as parameters form a line that would intersect with a line formed by connecting the coordinates from this ShadePoints coordinates to the coordinates from ShadePoint::connectedPoint.

Parameters
x1The horizontal position of the first endpoint.
y1The vertical position of the first endpoint.
x2The horizontal position of the second endpoint.
y2The vertical position of the second endpoint.
cXServes as an output parameter. If a collision occurs, the value will represent the horizontal position of the collision. Otherwise, it is unmodified.
cYServes as an output parameter. If a collision occurs, the value will represent the vertical position of the collision. Otherwise, it is unmodified.
Returns
true if the lines intersected. false otherwise.
