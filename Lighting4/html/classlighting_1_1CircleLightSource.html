void lighting::CircleLightSource::updateCastPoints ( CircleShadePoint *  updatePoint)
protectedvirtual

When the end of the line of an alphaPoint is reached, this function will check if any ShadePoints at the same radian as the endpoint are valid. If any are valid, it will return the line that will be closest to the origin using the output parameters.

<par> Will add the updatePoint to castPoints if the ShadePoint::connectedPoint has a greater angle. Will remove updatePoint 's ShadePoint::connectedPoint from castPOints if updatePoint has a radian greater than the ShadePoint::connectedPoint. Will account for lines crossing radian zero. </par>

Parameters
updatePointThe CircleShadePoint to check if adding or removing is needed.
